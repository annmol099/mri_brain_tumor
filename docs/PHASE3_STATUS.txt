## 🚀 HOW TO USE

### Quick Start
```bash
# Complete evaluation with all features
python evaluate.py --model models/best_model.h5 --data data/processed

# With training history for curves
python evaluate.py --model models/best_model.h5 --data data/processed \
    --history logs/training_history.json

```

## 💡 TIPS FOR USING EVALUATION

1. **Always include training history** for better insights:
   ```bash
   python evaluate.py --model MODEL --data DATA --history HISTORY
   ```

2. **Check Grad-CAM** to validate model reasoning:
   • Red/yellow regions should highlight tumors
   • Helps identify if model is learning correctly

3. **Review confusion matrix** to find problem classes:
   • High off-diagonal values indicate confusion
   • Focus improvement on those classes

4. **Compare per-class metrics**:
   • Identifies which tumor types are harder to classify
   • Guides data collection and augmentation strategies

5. **Use custom output dirs** for experiments:
   ```bash
   python evaluate.py --model MODEL --data DATA --output results/exp_1
   ```
